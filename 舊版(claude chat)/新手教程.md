# 🚀 菲律賓話題搜尋應用 - 完整新手教程

## 第一部分：環境準備（15分鐘）

### 步驟 1️⃣：安裝 Node.js

**什麼是 Node.js？**
- Node.js 是讓你能在電腦上運行 JavaScript 的工具
- 它會同時安裝 `npm`（Node Package Manager），這是安裝程式庫的工具

**安裝步驟：**

1. 進入 https://nodejs.org
2. 你會看到兩個版本：
   - **LTS**（推薦）= Long Term Support，更穩定
   - **Current** = 最新功能，但可能不穩定
   - 👉 **選擇 LTS 版本下載**
3. 下載完成後，點兩下安裝檔案
4. 一直按「下一步」直到完成

**驗證安裝成功：**

打開終端（Windows 用「命令列」，Mac 用「終端」）輸入：
```bash
node --version
npm --version
```

應該會看到版本號，例如：
```
v18.17.0
9.6.7
```

---

### 步驟 2️⃣：建立專案資料夾

在你喜歡的位置（例如「桌面」）建立一個資料夾，名稱為 `trending-ph`

然後在終端中進入該資料夾：
```bash
cd ~/Desktop/trending-ph
# 或者 Windows 用戶：
cd C:\Users\YourName\Desktop\trending-ph
```

---

### 步驟 3️⃣：初始化 React 專案

執行以下命令建立一個 React 專案：

```bash
npx create-react-app .
```

這會自動下載和設置所有必要的檔案。**需要 5-10 分鐘，請耐心等待。**

等待完成後，你會看到：
```
Success! Created app at your-path
```

---

### 步驟 4️⃣：替換應用程式碼

1. 打開 `src` 資料夾
2. 找到 `App.jsx` 檔案
3. 用提供的 `TrendingPH-App.jsx` 裡的完整代碼**替換** `App.jsx` 的內容

或者，複製 `TrendingPH-App.jsx` 的內容，貼到 `src/App.jsx` 中

---

### 步驟 5️⃣：安裝必要的套件

進入終端，執行：
```bash
npm install lucide-react
```

這會安裝圖標庫（用來顯示漂亮的圖示）

## 第四部分：代碼結構解析（進階）

### 文件結構

```
trending-ph/
├── public/
├── src/
│   ├── App.jsx          👈 主應用程式（你複製的代碼在這）
│   ├── App.css
│   ├── index.js
│   └── ...
├── package.json         👈 專案設定檔
└── ...
```

### 代碼的三個主要部分

#### 1️⃣ **狀態管理（State）**
```javascript
const [searchQuery, setSearchQuery] = useState('');
// 當使用者輸入時，searchQuery 改變
// setSearchQuery 是更新它的方法
```

**為什麼重要？**
- State 就是「應用的記憶」
- 當 State 改變時，網頁自動重新繪製（render）

#### 2️⃣ **數據（Data）**
```javascript
const mockTrendingData = {
  entertainment: [...],
  gaming: [...]
}
```

**現在的狀態：**
- 使用假數據（Mock Data）讓你看到效果
- 之後會連接真正的 API

#### 3️⃣ **UI 組件**
```javascript
<div>
  <input />      {/* 搜尋框 */}
  <button />     {/* 搜尋按鈕 */}
  <div>          {/* 話題卡片 */}
</div>
```

**樣式系統：**
- 使用 `style={{ }}` 直接定義樣式（叫 Inline Styles）
- 所有顏色都用 CSS 變數（`var(--color-*)`），自動支援深淺主題

## 第五部分：修改和擴展

### 簡單修改 - 改變應用名稱

**檔案：`src/App.jsx`**

找到這一行：
```javascript
<h1 style={{ margin: 0, fontSize: '28px', fontWeight: '500' }}>
  菲律賓 話題搜尋
</h1>
```

改成你喜歡的名稱：
```javascript
<h1 style={{ margin: 0, fontSize: '28px', fontWeight: '500' }}>
  🇵🇭 菲律賓熱門話題
</h1>
```

**保存檔案，網頁會自動更新！**

---

### 中級修改 - 添加新的話題

找到 `mockTrendingData` 物件：

```javascript
const mockTrendingData = {
  entertainment: [
    {
      id: 1,
      title: 'It\'s Showtime - GMA Network',
      category: 'Entertainment',
      trend: '+45%',
      views: '2.3M',
      source: 'Facebook',
      date: '今天',
      description: '菲律賓最受歡迎的綜藝節目話題爆紅',
      image: '🎬'
    },
    // 👇 在這裡添加新項目
    {
      id: 2,
      title: '新話題標題',
      category: 'Entertainment',
      trend: '+30%',
      views: '1.2M',
      source: 'Twitter',
      date: '今天',
      description: '新話題的描述',
      image: '🌟'
    }
  ]
}
```

每個項目都是一個物件，有這些屬性：
- `id` - 唯一識別碼（必須不重複）
- `title` - 話題標題
- `category` - 分類（Entertainment 或 Gaming）
- `trend` - 熱度趨勢
- `views` - 討論量
- `source` - 來源（Facebook, Twitter 等）
- `date` - 日期
- `description` - 描述
- `image` - emoji 圖示

## 第六部分：連接真正的 API（進階）

### 現在用的是假數據，如何換成真實數據？

#### 選項 1：Google Trends API

```javascript
// 在搜尋函數裡添加：
const response = await fetch('https://api.google.com/trends?query=Philippines');
const data = await response.json();
```

**需要做的：**
1. 申請 Google API 金鑰
2. 解析返回的 JSON 數據
3. 轉換成你的格式

#### 選項 2：Twitter/X API

**需要做的：**
1. 申請 Twitter Developer 帳號
2. 獲得 API 金鑰和密鑰
3. 用 `axios` 或 `fetch` 調用 API

```javascript
const response = await fetch('https://api.twitter.com/2/tweets/search/recent?query=Philippines', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```

#### 選項 3：新聞 RSS 源

```javascript
// 使用 RSS 解析庫
const RSS = require('rss-parser');
const parser = new RSS();
const feed = await parser.parseURL('https://rappler.com/feed');
```

**這個最簡單！** 不需要認證，直接讀取新聞

## 第七部分：常見問題

### Q: 改了代碼，網頁沒更新？
**A:** 
1. 確保保存檔案（Ctrl+S 或 Cmd+S）
2. 查看終端，是否有錯誤訊息
3. 重新整理瀏覽器（F5）

### Q: npm install 很慢？
**A:** 
- 可能是網路問題
- 重新執行一次
- 或改用 `yarn`（更快的套件管理工具）

### Q: 想在手機上看？
**A:**
1. 終端顯示的 Local 地址改成你電腦的 IP
   ```
   ipconfig          # Windows
   ifconfig          # Mac
   ```
2. 在手機瀏覽器輸入：`http://YOUR_IP:3000`

### Q: 怎樣部署到網際網路讓別人看到？
**A:** 使用免費平台，例如：
- **Vercel** - 最簡單，專門為 React 設計
- **Netlify** - 也很好用
- **Heroku** - 功能較多

## 第八部分：部署到網際網路（Vercel）

在終端執行：

這會建立一個優化版本，檔案大小更小，速度更快。

```bash
npm install -g vercel

```

1. 選擇帳號（如果沒有則建立免費帳號）

```
https://trending-ph.vercel.app
```

任何人都可以透過這個 URL 使用你的應用！
